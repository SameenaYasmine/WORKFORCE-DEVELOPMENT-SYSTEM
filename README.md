**Workforce Development System**
A web-based Workforce Development System built using the MERN stack to manage training programs, track student progress, automate certification, and generate analytics reports.

** Tech Stack**

**Frontend**
React.js
HTML5, CSS3
JavaScript (ES6+)

**Backend**
Node.js
Express.js
Database
MongoDB

**Tools**

VS Code
GitHub
Postman

**Features**
Role-based authentication (Admin, Instructor, Student)
Course creation and management
Student enrollment and progress tracking
Skill assessments
Automated certificate generation
Dashboard with analytics and reports
